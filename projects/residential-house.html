Project Overview

This ongoing project focuses on the structural design of a residential wood-frame house. I was provided the architectural drawings and am using them as the basis for developing the structural layout, load calculations, framing plans, and final drawing package.

Current Status

This project is still in progress, and this page will continue to be updated as more drawings, calculations, and design documents are completed.

In Progress: Refining the structural drawing set and preparing additional documentation.
Next Step: Add more project images, calculation references, and drawing sheets as the project develops.

Design Process

The process began with reviewing the architectural drawings and understanding the layout of the home. From there, I developed the structural approach by considering load paths, framing direction, support locations, and how the roof, walls, and floor system work together.


Load calculations were completed to guide the framing decisions, and AutoCAD was used to create framing plans that communicate the structural layout clearly.
