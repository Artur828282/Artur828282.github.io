Structural Design Calculators

Custom Excel-based engineering tools developed to automate structural and seismic design calculations using ACI 318, ASCE 7-22, and IBC 2024 references.

Project Overview

This project includes three Excel-based engineering calculators created to support structural design and seismic calculation workflows. The calculators were developed to organize inputs, automate repeated calculation steps, reference applicable code provisions, and clearly display final design checks.


The goal was to build practical tools that make structural calculations easier to review, repeat, and verify. Each calculator was designed with clear inputs, intermediate results, code references, and pass/fail checks.

Calculator 1: Concrete Basement Wall

This calculator evaluates a residential concrete basement wall using soil pressure, surcharge loading, moment demand, shear demand, reinforcement requirements, and capacity checks. It includes design references from IBC 2024, ACI 318-19, and ASCE 7-22.

Calculator 2: Seismic Loading Criteria

This calculator organizes ASCE 7-22 seismic loading criteria, including site class, risk category, seismic importance factor, spectral acceleration values, seismic design category, response coefficients, and seismic response coefficient checks.

Calculator 3: Vertical Distribution of Lateral Forces

This calculator determines the vertical distribution of lateral seismic forces using ASCE 7-22 procedures. It includes story force calculations, diaphragm force calculations, distribution factors, and final force checks by level.
